Social Stopwatch Options

Set Default Alarms

Use the inputs below to set Default Alarms. Default Alarms will automatically create an alarm using the time you input whenever you navigate to a url that contains what you typed.
For example: typing reddit.com in the URL box, then setting H to 1, M to 20, and S to 0, will make Social Stopwatch create an alarm that's 1hr and 20min long for every tab that goes to a website with reddit.com in the URL.
